# 品络企业网站系统

#### 介绍
品络企业网站系统，拥有企业网站常用的模块功能：企业简介模块、联系我们模块、新闻(文章)模块、产品模块、图片模块、招聘模块、在线留言、反馈系统、在线交流、友情链接、网站地图、栏目管理、网站碎片、管理员与权限管理等等，所有模块的分类均支持无限级别的分类，可拓展性强大。特别是栏目管理系统、网站碎片管理系统，可以灵活组合出各种不同的页面和应用。
